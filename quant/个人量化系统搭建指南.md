## 资金规划与风险控制

| 资金量 | 建议用途 | 风控要求 |
|--------|----------|----------|
| **1-5 万** | 模拟盘 + 极小实盘（单笔 <1000 元） | 专注学习和验证系统，可接受完全亏损 |
| **5-10 万** | 1-2 个简单策略（动量、均值回归），单策略 <5000 | 单策略最大回撤 <15%，严格执行止损 |
| **10-20 万** | 多策略组合，开始仓位管理 | 组合最大回撤 <20%，考虑行业/风格分散 |
| **20 万+** | 全系统实盘，考虑融资、多资产、自动化 | 严格风控规则，定期复盘归因 |

**⚠️ 重要原则：**
- 实盘前至少 6 个月模拟测试 + 样本外验证
- 单策略最大回撤控制在 15% 以内
- 不要用生活必需资金做量化交易
- 从小资金开始，逐步增加

## 量化研究核心能力

### 因子挖掘
- **传统因子：**
  - 动量（Momentum）：过去 N 日收益率
  - 反转（Reversal）：短期反转效应
  - 价值（Value）：PE、PB、股息率
  - 质量（Quality）：ROE、ROA、资产负债率
  - 成长（Growth）：营收增长率、净利润增长率
  - 波动率（Volatility）：历史波动率
  - 流动性（Liquidity）：换手率、成交额

- **机器学习因子：**
  - 非线性组合（XGBoost、LightGBM）
  - 时序预测（LSTM、Transformer）
  - 特征交叉与自动挖掘

### 因子有效性检验
- **IC（信息系数）：** 因子值与未来收益率的相关性
  - |IC| > 0.05 有意义
  - |IC| > 0.1 较好
  - 稳定性：时间序列上的标准差
- **IR（信息比率）：** 平均 IC / IC 标准差
  - IR > 0.5 可用
  - IR > 1 较好
- **分组回测：** 按因子值分 5-10 组，观察单调性
- **换手率：** 因子稳定性，换手过高成本高
- **衰减周期：** 因子有效期（日频/周频/月频）

### 因子组合
- 标准化（Z-Score）
- 正交化（去除共线性）
- 加权合成：
  - 等权
  - IC 加权
  - 机器学习组合（回归、排序）

### 回测陷阱
- **幸存者偏差：** 只用当前存活股票，忽略已退市
- **前视偏差：** 使用未来数据
- **过拟合：** 参数优化过度，样本外失效
- **交易成本：** 忽略手续费、滑点、冲击成本
- **流动性约束：** 无法成交、涨跌停、停牌
- **复权处理：** 除权除息、股本变动

### 策略类型
- 多因子选股
- 统计套利
- 配对交易
- 事件驱动
- 日内高频
- CTA 趋势

## 从模拟到实盘的关键路径

```
阶段 1：单因子研究（1-2 个月）
    ↓
    输出因子报告（IC、分组收益、换手）

阶段 2：多因子组合 + 回测（2-3 个月）
    ↓
    实现完整回测框架，加入交易成本

阶段 3：模拟盘验证（3-6 个月）
    ↓
    实时计算因子，模拟下单，跟踪偏差

阶段 4：极小资金实盘（6-12 个月）
    ↓
    单策略，严格风控，记录每笔交易

阶段 5：多策略 + 自动化（12 个月+）
    ↓
    组合优化、仓位管理、自动化交易
```

### 第 1 个月：金融基础 + 工具链搭建
- [ ] 📖 读书：《漫步华尔街》《打开量化投资的黑箱》《主动投资组合管理》第 1-3 章

### 第 2-3 个月：单因子研究与回测框架
- [ ] 📖 读书：《量化投资：策略与技术》《因子投资：方法与实践》第 1-4 章
- [ ] 🔬 因子：实现 5 个经典因子（动量 20 日、反转 5 日、PE、ROE、波动率）
- [ ] 📈 回测：写第一个向量化回测（月度调仓、等权组合、手续费 0.03%）
- [ ] 🎯 产出：单因子报告（IC 均值、IR、分组累计收益、最大回撤、换手率）

### 第 4-6 个月：多因子组合 + C++ 原型
- [ ] 📖 读书：《因子投资：方法与实践》第 5-9 章、《金融计量学：时间序列分析视角》
- [ ] 🧮 组合：因子标准化、正交化、等权/IC 加权合成
- [ ] ⚡ C++：用 Eigen 实现因子计算核心（如移动平均、标准差），对比 Python 性能
- [ ] 🎯 产出：多因子策略回测报告（年化收益、夏普、最大回撤、月度胜率）

### 第 7-9 个月：风险模型 + 组合优化
- [ ] 📖 读书：《主动投资组合管理》第 4-10 章、《Risk and Asset Allocation》
- [ ] 🛡️ 风控：计算组合协方差矩阵、跟踪误差、行业暴露
- [ ] ⚖️ 优化：实现最小方差组合、风险平价、带约束的均值-方差优化（cvxpy）
- [ ] 🎯 产出：不同优化方法对比实验 + 风险归因分析

### 第 10-12 个月：模拟盘 + 实盘准备
- [ ] 🤖 模拟：接入实时行情（Tushare Pro/券商 API），每日计算因子和信号
- [ ] 📊 监控：搭建监控看板（净值曲线、持仓、因子暴露、风险指标）
- [ ] 📝 文档：写策略说明书（逻辑、参数、风控规则、历史表现）
- [ ] 🎯 产出：6 个月模拟盘记录 + 实盘交易计划（资金分配、止损规则）

### 第 13-18 个月：极小资金实盘 + 迭代
- [ ] 💰 实盘：1-2 个策略，单策略资金 <10% 总资产，手动下单或半自动
- [ ] 📊 复盘：每周复盘交易记录，对比模拟与实盘偏差
- [ ] 🔍 归因：分析收益来源（因子贡献、行业配置、个股选择）
- [ ] 🎯 产出：实盘交易日志 + 季度策略评估报告

### 第 19-24 个月：系统化与自动化
- [ ] ⚡ C++ 低延迟：重构核心计算模块（因子、信号）为 C++ 库，Python 调用
- [ ] 📡 数据管道：搭建自动数据更新 + 因子计算 + 信号生成流水线
- [ ] 🤖 自动化：实现自动下单（券商 API）、风控检查、异常告警
- [ ] 🎯 产出：个人量化交易系统（Python 前端 + C++ 核心 + 监控后台）

## 项目清单

### 1. 因子研究实验室（Python）
- **目标：** 自动下载数据 → 计算 10+ 因子 → 输出因子报告（IC/IR/分组收益/换手）→ 可视化
- **技术栈：** pandas, numpy, sklearn, matplotlib/pyecharts, Tushare/AkShare
- **产出：** 因子有效性分析报告、可视化图表

### 2. 向量化回测引擎（Python）
- **目标：** 支持多因子、月度调仓、手续费/滑点、行业/市值中性、绩效分析
- **技术栈：** pandas（向量化计算）、numpy、cvxpy（优化）、pyecharts（可视化）
- **产出：** 回测报告（净值曲线、年化收益、夏普、最大回撤、换手率）

### 3. C++ 因子计算库（高性能核心）
- **目标：** 用 Eigen 实现移动窗口统计（均值、标准差、相关性）、技术指标、因子正交化
- **技术栈：** C++17, Eigen, Arrow/Parquet, pybind11（Python 绑定）
- **产出：** C++ 库 + Python 绑定，性能对比报告

### 4. 模拟交易监控看板
- **目标：** 实时显示持仓、净值、因子暴露、风险指标、交易信号历史
- **技术栈：** FastAPI（后端）、ECharts（前端）、WebSocket（实时推送）
- **产出：** Web 监控看板，支持实时更新

### 5. 组合优化工具箱
- **目标：** 比较等权、最小方差、风险平价、Black-Litterman、带约束优化的表现
- **技术栈：** cvxpy, numpy, pandas, matplotlib
- **产出：** 不同优化方法对比实验、风险归因分析

## 必读书单

### 📖 金融基础（必读）
1. **《漫步华尔街》** - 伯顿·马尔基尔
   - 了解市场基本规律，破除技术分析迷信
   - 重点：有效市场假说、指数投资

2. **《打开量化投资的黑箱》** - 里什·纳兰
   - 量化对冲基金工作流程揭秘
   - 重点：量化系统组成、风险管理

### 📖 因子投资（核心）
3. **《因子投资：方法与实践》** - 石川
   - 中文因子圣经，必读
   - 重点：因子挖掘、有效性检验、组合构建

4. **《量化投资：策略与技术》** - 丁鹏
   - 国内量化入门经典
   - 重点：策略分类、回测方法

### 📖 组合管理（进阶）
5. **《主动投资组合管理》** - Grinold & Kahn
   - 量化组合圣经
   - 重点：信息比率、组合优化、风险模型

6. **《Risk and Asset Allocation》** - Attilio Meucci
   - 风险与资产配置经典
   - 重点：风险度量、Black-Litterman

### 📖 金融计量（数学）
7. **《金融计量学：时间序列分析视角》** - 张成思
   - 时间序列基础

8. **《Time Series Analysis》** - James D. Hamilton
